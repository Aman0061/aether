# Aether - Luxury Interior Design

React приложение для портфолио дизайна интерьеров, переписанное с чистого HTML/CSS/JS на React с использованием Vite.

## Технологии

- **React 18** - UI библиотека
- **React Router DOM** - Роутинг для навигации между страницами
- **Vite** - Сборщик и dev-сервер
- **Tailwind CSS** - Утилитарный CSS фреймворк
- **Lucide React** - Иконки

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist`

## Предпросмотр продакшен сборки

```bash
npm run preview
```

```
aether/
├── public/
│   └── images/      # Статические изображения

│   │   ├── Hero.jsx         # Hero секция

│   │   ├── Services.jsx     # Секция услуг
│   │   ├── Footer.jsx       # Футер
│   │   ├── ProjectsPage.jsx # Страница всех проектов
│   │   ├── StudioPage.jsx   # Страница студии
│   │   ├── ServicesPage.jsx # Страница услуг
```

## Роутинг

Приложение использует React Router для навигации между страницами:

- `/` - Главная страница (Home)
- `/projects` - Все проекты
- `/studio` - О студии
- `/services` - Услуги
- `/journal` - Журнал
- `/contact` - Контакты
